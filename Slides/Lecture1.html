Basic definitions

Graphs can encode connections

A graph $\Gamma$ consists of:

  • A set $V(\Gamma)$ of vertices (things)
  • A set $E(\Gamma)$ of edges (connections)

Each edge $e\in E(\Gamma)$ connects two vertices in $V(\Gamma)$.

Examples

More Definitions

  • If there is an edge between two vertices, we say they are adjacent
  • If an edge $e$ connects to a vertex $v$, we say they are incident
  • The degree of a vertex is the number of edges incident to it

Some features are not always wanted:

  • A loop is an edge connecting a vertex to itself
  • There could be multiple edges between the same two vertices

      • Graphs without loops or multiple edges are called simple

Subgraphs

A subgraph $S$ of $\Gamma$ is a graph $S$ where

  • $V(S)\subset V(\Gamma)$
  • $E(S)\subset E(\Gamma)$

This implies the vertices at the end of each edges of $S$ to be in $S$
